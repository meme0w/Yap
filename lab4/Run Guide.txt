Для запуска прописать в директории "cargo run -- <URL>"

URL - ссылка для скачивания файла
Ссылка не должна содержать знак '&', т.к. он зарезирвирован системой.

Название и расширение файда берутся из URL(находятся после последнего знака '/').

Лучше брать файлы размера от 100 мб, чтобы успеть пронаблюдать вывод количества скачанных байт.
